# Folios — Said HAMMANE Portfolio

Personal portfolio built with **Laravel 11** to present my profile, experience, and selected BI / Data case studies.

## Positioning

This portfolio is built to support applications for roles such as:
- BI Analyst
- Data Analyst
- Reporting Analyst
- Junior Data Engineer
- BI Developer
- Odoo / ERP data-related roles

## Local setup

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run build
php artisan serve
```

If using the Laravel dev workflow:

```bash
composer run dev
```

## Important note

This repository contains the portfolio application itself. The strongest part of the portfolio is not the framework choice — it is the **clarity of positioning**, **real BI/data case studies**, and **business-oriented project presentation**.
